# node-js-learning

## Install

To install all dependencies you need run command only on root
```shell
npm ci
```

## Run

**Pre setup step**. Create `.env` from `.env.example`.
File `.env` stores ports for servers and other envs
```shell
cp .env.example .env
```

**Run server**
```shell
npm run start
```

**Run server in dev mode**
```shell
npm run dev
```
